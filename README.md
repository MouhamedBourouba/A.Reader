# A.Reader

A.Reader is an elegant and intuitive Android app designed to enhance your reading experience. It serves as your personal companion to manage the books you are reading, helping you stay organized, engaged, and immersed in your literary adventures.

## App Features:
- Seamlessly organize and track the books you are currently reading, have read, or want to read.
- Personalize your reading experience with features like bookmarking, highlighting, and note-taking.
- Get comprehensive book information and covers, by integrating with the Google Books API.
- Rate and review books to keep a record of your thoughts and share recommendations with others.
- Enjoy the convenience of cloud storage for seamless synchronization of your reading progress across multiple devices.
- Secure your personal data with user authentication and account management using JWT authentication.
- Immerse yourself in a modern and visually appealing user interface crafted with Jetpack Compose.

## Technologies Used
- Jetpack Compose: Android's modern toolkit for building native UI with a declarative and efficient approach.
- Ktor: A powerful backend framework for building robust and scalable Android apps.
- MVVM Architecture: A design pattern that ensures separation of concerns and facilitates efficient app development.
- Retrofit: A popular HTTP client for Android that simplifies networking and API communication.
- JWT Authentication: Secure user login and authentication mechanism for protecting user data and privacy.
- Google Books API: Integration with the API to fetch comprehensive book information, covers, and reviews.
